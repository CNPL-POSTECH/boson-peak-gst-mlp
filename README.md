# Data and Analysis Scripts for “Study of boson peak in amorphous Ge-Sb-Te using machine learning potentials”

This repository contains the machine-learning potential, input files, representative scripts, and numerical data used in the article:

**P. Park and S.-H. Jhi, “Study of boson peak in amorphous Ge-Sb-Te using machine learning potentials,” Physical Review B (2026).**

## 01.Machine_Learning_Potential

This directory contains the dataset and trained MACE potentials used in the molecular-dynamics simulations.

### `Data_set/`

This folder contains the training and validation data used to construct the GST machine-learning potential.

* `mace_run-123.log`
  Log file from the MACE training run.

* `train.xyz`
  Training dataset in extended XYZ format.

* `validate.xyz`
  Validation dataset in extended XYZ format.

### `Potential/`

This folder contains the trained MACE potential files.

* `mace_swa.model`
  MACE model used in the ASE-based molecular-dynamics simulations.

* `mace_swa.model-lammps.pt`
  MACE model converted for use with LAMMPS.

## 02.Input

This directory contains representative input files and scripts used for molecular dynamics, elastic-constant calculations, and dynamical-structure-factor analysis.

### `Dynamical_Structure_Factors/`

This folder contains scripts used to calculate and analyze dynamical structure factors.

* `dyna.py`
  Script for calculating dynamical structure factors using the `dynasor` package.

* `fit_DHO_vs_q.py`
  Post-processing script used to fit the dynamical structure factor spectra with a damped harmonic oscillator model.

* `ir_thz_split_fit.py`
  Post-processing and plotting script used to analyze the Ioffe-Regel crossover frequency.

### `Elastic_constant/`

This folder contains input files and scripts used to calculate elastic constants.

### `MD/`

This folder contains the ASE molecular-dynamics script.

* `ase_md.py`
  ASE-based molecular-dynamics script used to perform the melting, quenching, heating, and crystallization procedure.

## 03.Result

This directory contains the figures and numerical data supporting the article.

### `Figure/`

This folder contains figures corresponding to the results presented in the article.

### `Raw_Data/`

This folder contains numerical data used for the analysis.

* `Boson_Peak/`
  Boson peak frequencies, intensities at different crystallization times.

* `Ioffe_Regel/`
  Dynamical structure factor fitting results and Ioffe-Regel crossover data.

* `Structure/`
  Dump files containing atom-resolved structural and mechanical quantities. These files include atomic shear modulus, reduced vibrational density of states, coordination number, order parameter, and Voronoi cell volume.

The quantities included in the structure dump files are:

* G: atomic shear modulus
* RVDOS: atomic reduced vibrational density of states intensity
* CN: coordination number
* O: structural order parameter
* voro: Voronoi cell volume

## Notes on full trajectories

Full molecular dynamics trajectories are not included because of their large file size. Instead, this repository provides representative input files, trained potentials, processed numerical data, and structural dump files required to reproduce the main analyses and figures.

## Software

The main calculations and analyses used the following software packages:

* MACE
* ASE
* dynasor
* LAMMPS
